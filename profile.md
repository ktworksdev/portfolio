# 🚀 Portfolio

## JWT Task Manager App

### 概要

JWT認証を採用したタスク管理アプリです。

Node.js・Express・SQLiteを使用し、ユーザー登録・ログイン機能、JWT認証、タスクのCRUD操作を実装しました。

フロントエンドとバックエンドを分離した構成でREST APIを設計し、本番環境へのデプロイと動作確認まで実施しています。

### 主な機能

- ユーザー登録
- ログイン・ログアウト
- JWT認証
- タスク一覧取得
- タスク追加
- タスク編集
- タスク削除
- タスク完了・未完了切替
- ユーザーごとのタスク管理
- REST API
- レスポンシブ対応

### 使用技術

- HTML
- CSS
- JavaScript
- Node.js
- Express
- SQLite
- JWT
